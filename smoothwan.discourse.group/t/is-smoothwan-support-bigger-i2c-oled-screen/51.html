Is smoothwan support bigger I2C Oled screen?

Help

Hi, may i know if smoothwan support bigger I2C Oled screen. If yes, can you advise the model?
thanks

TalalM April 10, 2023, 2:19am 2

Hi, unfortunately OLED support has been dropped since the displays have poor longevity.
Next release includes MQTT reporting for all VPNs, and a display like M5 stack can be used via USB/Wi-Fi or other displays.
For the older version, yes they do support larger displays as long as they use the same chip “ssd1306”.
